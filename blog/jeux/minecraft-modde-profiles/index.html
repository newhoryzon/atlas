    Profiles Minecraft Modded

    Ci-après, la liste des mods considéré comme nécessaire pour une partie, d'abord les mods d'optimisation qui n'impact pas le gameplay ni le client du jeu. Il est fortement recommandé de démarrer toute partie en utilisant ces mods.

    Légendes :

    ☑️ : FABRIC uniquement ✅ : FABRIC et FORGE 🟪 : FORGE uniquement ❕ : Disponible NEOFORGE 📦 : Installation manuelle (lien vers le fichier)

    ✔️ : Disponible sur FORGE 🔶 : Uniquement sur CurseForge
    ❌ : Indisponible sur FORGE ➰ : Remplacement pour FORGE/NEOFORGE
